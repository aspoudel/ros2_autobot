UWA's Mobile Robots(AUTO4508) project demonstrating the implementation of autonomous driving on a Pioneer 3AT Mobile Robot platform with various hardwares like IMU, GPS, Camera, Lidar, etc.
The robot uses ROS2 for brains and can be coded to run autonomously in any environment to map and collect visual data of the environment. It can be used in environments too
hostile for humans to be in for autobotmous driving, navigation and mapping.
